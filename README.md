# Rams Pit Wall

> Lights out and away we go — a personal Formula 1 dashboard built as a single-file web experience for the 2026 season.

## Overview
Rams Pit Wall is a Formula 1 themed personal dashboard that brings race-weekend energy into one browser tab. It presents the next Grand Prix countdown, season calendar, driver standings, constructor standings, podium snapshots, and paddock-style updates in a bold editorial layout inspired by motorsport coverage.

This project is built as a single HTML file, which makes it easy to run, share, and deploy. Open it locally in a browser or publish it with GitHub Pages for a live portfolio-style showcase.

## Features
- Live countdown to the next Grand Prix.
- 2026 season calendar in a clear race-by-race layout.
- Driver standings and constructor standings in one dashboard.
- Podium and paddock-inspired sections for a magazine-like feel.
- Fully front-end and lightweight, with no backend required.
- Fast to deploy because the project is contained in one main HTML file.

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages for hosting

## Run Locally
1. Download or clone the repository.
2. Open `rams_pit_wall.html` in any modern browser.
3. Enjoy the dashboard like a race engineer watching the timing screen.

## Deploy on GitHub Pages
1. Push the project to a GitHub repository.
2. Go to **Settings** > **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and save.
5. GitHub will generate a public link for the site.
